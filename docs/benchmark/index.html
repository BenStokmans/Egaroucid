Edaxとの対戦

世界最強とも言われるオセロAI、Edaxとの対戦結果です。

初手からの対戦では正確な強さがわからないと考えたため、最終的に引き分けとなった棋譜を使って途中まで打ち、そこから対戦させました。

途中からの対戦では、その時点で勝敗または引き分けが決しているので、両者の強さが全く同じであれば勝敗の数は同じになります。勝敗の偏りを見ることで強さがわかります。

bookは双方使用していません。
